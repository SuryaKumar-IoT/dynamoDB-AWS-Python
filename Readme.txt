1.Install AWS CLI in your system

2.Install Python boto3 package in your system

pip install boto3

3.configure your aws cli

using command prompt

4.open command prompt

type

aws configure

it will ask you to enter 

aws_access_key_id=""
aws_secret_access_key=""
region_name=""
default_output="json"

enter these commands it will configure your aws resource 

5.to create table in dynamodb run creat_table.py

6.to insert items in to table you can run insert_items.py

7.to read data from table you can run read_table.py
